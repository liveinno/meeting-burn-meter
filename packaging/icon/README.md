# Иконка «Костёр» — генератор

Иконка приложения (`Kostyor.App/Assets/kostyor.ico`) — не рисованный вручную блоб, а
результат детерминированного генератора на GDI+. Это позволяет пересобрать её в любой момент
и держать под контролем версий исходник, а не только бинарник.

## Что рисуется

Точная реализация **финального макета дизайнера** (вариант 1b,
`docs/design/Иконка Костёр - финал.dc.html`): костёр — пламя (снизу `#fde047` →
`#fb923c` → верх `#f43f5e`) с горячим ядром (`#fffbeb` → `#fde047`) над двумя тлеющими
поленьями, на тёмной радиальной плитке (`#1d222c` → `#0a0c11`) со свечением и углями.
Пути безье и градиенты — 1:1 из SVG дизайнера (viewBox 512), масштабируются под каждый размер.

## Как пересобрать

```bash
cd packaging/icon
dotnet run -c Release -- .            # выдаст kostyor.ico + preview_256.png / preview_64.png
cp kostyor.ico ../../Kostyor.App/Assets/kostyor.ico
```

Многоразмерный `.ico`: 16/24/32/48/64/128 — 32-bpp DIB (совместимо с треем и
`IconBitmapDecoder`, см. BUGS.md), 256 — PNG-кадр. Проект **не** входит в `Kostyor.sln`
(отдельный тул), поэтому обычная сборка решения его не трогает.
